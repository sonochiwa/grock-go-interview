# 01 — Основы Go

Этот раздел охватывает фундаментальные концепции языка Go, которые составляют основу любого собеседования на позицию Middle Go-разработчика. Каждая тема содержит теорию, примеры кода, описание внутреннего устройства и типичные вопросы интервью.

## Содержание

| # | Тема | Описание |
|---|------|----------|
| 01 | [Типы и переменные](01-types-and-variables.md) | Базовые типы (`int`, `float`, `bool`, `byte`, `rune`), нулевые значения, приведение типов, алиасы vs определения типов, константы и `iota` |
| 02 | [Указатели](02-pointers.md) | Семантика указателей, `&` и `*`, `nil`-указатели, стек vs куча, `unsafe.Pointer` |
| 03 | [Структуры](03-structs.md) | Определение структур, встраивание (композиция), методы (value vs pointer receiver), анонимные структуры, сравнение, выравнивание и паддинг |
| 04 | [Слайсы](04-slices.md) | Заголовок слайса, `make` vs литерал, механика `append`, алгоритм роста, трюки со слайсами, утечки памяти |
| 05 | [Карты (maps)](05-maps.md) | Использование map, случайный порядок итерации, `nil` map, конкурентный доступ, рост map |
| 06 | [Строки](06-strings.md) | Внутреннее устройство строк, руны vs байты, UTF-8, `strings.Builder`, стоимость конвертации |
| 07 | [Функции](07-functions.md) | Функции первого класса, замыкания, `defer`, вариативные функции, именованные возвраты, `init()` |

## Как использовать

1. Читайте файлы последовательно — каждый следующий опирается на предыдущий
2. Обращайте внимание на секции **"Под капотом"** — именно эти знания отличают middle от junior
3. Прорабатывайте **"Частые вопросы на собеседованиях"** — формулируйте ответы вслух
4. Изучайте **"Подводные камни"** — это реальные ошибки из production-кода
