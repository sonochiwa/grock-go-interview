# Unique Elements

Напиши функцию `Unique[T comparable](s []T) []T`, которая возвращает слайс только с уникальными элементами, сохраняя порядок первого появления.

## Требования

- Функция должна быть generic с constraint `comparable`
- Порядок первого появления сохраняется
- Пустой слайс возвращает пустой слайс
- `nil` слайс возвращает `nil`

## Пример

```go
Unique([]int{1, 2, 2, 3, 1, 4}) // []int{1, 2, 3, 4}
Unique([]string{"a", "b", "a", "c"}) // []string{"a", "b", "c"}
```
