# Ping Pong

Напиши функцию `PingPong(n int) []string`. Запусти 2 горутины, которые через канал передают друг другу сообщения: `"ping"`, `"pong"`, `"ping"`, `"pong"`... `n` раз. Верни все сообщения в порядке отправки.

## Требования

- Используй 2 горутины и каналы для синхронизации
- Первая горутина отправляет `"ping"`, вторая — `"pong"`
- Всего `n` сообщений
- Если `n <= 0`, вернуть пустой слайс
- Сообщения возвращаются в порядке отправки

## Пример

```go
PingPong(4) // []string{"ping", "pong", "ping", "pong"}
PingPong(1) // []string{"ping"}
PingPong(0) // []string{}
```
