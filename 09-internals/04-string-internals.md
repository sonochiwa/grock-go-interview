# Внутреннее устройство строк

## Структура

```go
// Строка = 16 байт на 64-bit
type stringHeader struct {
    Data unsafe.Pointer // указатель на байты (UTF-8)
    Len  int            // длина в БАЙТАХ (не рунах!)
}

// Нет поля Cap — строка НЕИЗМЕНЯЕМА
// Нет нуль-терминатора (в отличие от C)
```

## Ключевые свойства

- **Иммутабельность**: нельзя изменить байт — только создать новую строку
- **Подстрока разделяет память**: `s[2:5]` указывает на те же байты
- **string ↔ []byte**: требует копирования (кроме оптимизаций компилятора)
- **Сравнение**: побайтовое O(n), но если Data указывают на один адрес — O(1)

## Оптимизации компилятора

Компилятор НЕ копирует при:
```go
m[string(b)]        // map lookup
string(b) == "test" // сравнение
for _, r := range string(b) // range
"a" + string(b) + "c" // конкатенация
```

## unique.Handle (Go 1.23+)

```go
// Явный string interning
h := unique.Make("hello")
// Все unique.Make("hello") возвращают один и тот же Handle
// h1 == h2 → быстрое сравнение (по указателю)
```
