# Escape Analysis

Дано 6 пар функций — «медленная» (heap) и «быстрая» (stack) версия.

1. Для каждой пары объясни в комментарии: почему одна escapes, а другая нет
2. Напиши бенчмарки, доказывающие разницу
3. Проверь себя: `go build -gcflags="-m" .`

Задание не про реализацию — код уже написан. Нужно понять **почему** и написать бенчмарки.
