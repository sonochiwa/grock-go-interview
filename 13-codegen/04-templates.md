# Шаблоны для генерации

## Обзор

`text/template` используется для генерации Go кода из шаблонов.

```go
const tmpl = `// Code generated by mygen; DO NOT EDIT.
package {{.Package}}

{{range .Types}}
func (t {{.Name}}) Validate() error {
    {{range .Fields}}
    {{if .Required}}
    if t.{{.Name}} == {{.ZeroValue}} {
        return fmt.Errorf("{{.Name}} is required")
    }
    {{end}}
    {{end}}
    return nil
}
{{end}}`

// Данные для шаблона
type TemplateData struct {
    Package string
    Types   []TypeInfo
}

// Генерация
t := template.Must(template.New("gen").Parse(tmpl))
f, _ := os.Create("validate_gen.go")
t.Execute(f, data)
```

### Практический workflow

```
1. Написать шаблон (Go text/template)
2. Написать генератор (парсит AST → заполняет данные → рендерит шаблон)
3. Добавить //go:generate directive
4. Запустить go generate
5. Закоммитить сгенерированный файл
```
