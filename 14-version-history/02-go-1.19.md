# Go 1.19 (Август 2022)

## Ключевые изменения

### Ревизия Go Memory Model

Формальное описание обновлено, теперь соответствует C++ memory model:
- Atomic операции имеют sequential consistency (явно документировано)
- sync/atomic: новые типы `atomic.Int64`, `atomic.Bool`, `atomic.Pointer[T]`

```go
var counter atomic.Int64
counter.Add(1)
val := counter.Load()
```

### GOMEMLIMIT

```bash
GOMEMLIMIT=1GiB ./myapp
```

Мягкий лимит памяти. GC становится агрессивнее при приближении к лимиту. Решает проблему OOM при GOGC=off.

### Doc comments

Поддержка ссылок, списков и заголовков в документации:

```go
// # Heading
//
// [pkg.Func] — ссылка на символ
// [Go home page]: https://go.dev
```
