# Go 1.18 (Март 2022)

**Самый значимый релиз** после Go 1.0. Дженерики наконец-то в языке.

## Ключевые изменения

### Дженерики (Type Parameters)

```go
func Map[T, R any](s []T, f func(T) R) []R { ... }
```

- Type parameters для функций и типов
- Constraints через интерфейсы
- Type inference
- `any` как алиас для `interface{}`
- `comparable` constraint

### Fuzzing

```go
func FuzzParseJSON(f *testing.F) {
    f.Add(`{"key": "value"}`)
    f.Fuzz(func(t *testing.T, data string) {
        var m map[string]any
        json.Unmarshal([]byte(data), &m)
    })
}
```

Встроенная поддержка fuzzing в testing package.

### Workspaces (go.work)

```
go work init ./module1 ./module2
```

Работа с несколькими модулями одновременно.

### Другое

- Новый алгоритм роста слайсов (плавный вместо 2x→1.25x скачка)
- 20% улучшение производительности на ARM64
