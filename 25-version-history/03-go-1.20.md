# Go 1.20 (Февраль 2023)

## Ключевые изменения

### PGO (Preview)

Profile-Guided Optimization — компилятор использует CPU profile для оптимизации.

### comparable expansion

Comparable constraint теперь допускает типы, которые comparable в runtime (interfaces), даже если они не всегда comparable.

### errors.Join

```go
err := errors.Join(err1, err2, err3)
// errors.Is(err, err1) == true
```

### strings.Clone, bytes.Clone

```go
// Создаёт копию строки, освобождая backing array
small := strings.Clone(hugeSting[:10])
```

### Другое

- `context.WithCancelCause`
- Wrapping multiple errors (`%w` в fmt.Errorf можно несколько раз)
- Улучшения производительности компилятора
