# Go 1.21 (Август 2023)

## Ключевые изменения

### Новые built-in функции

```go
min(1, 2, 3) // 1
max(1, 2, 3) // 3
clear(m)     // очистить map или slice
```

### log/slog — структурированное логирование

```go
slog.Info("user created",
    "user_id", 42,
    "email", "alice@example.com",
)
// 2024/01/15 10:30:00 INFO user created user_id=42 email=alice@example.com
```

### Новые пакеты

- `slices` — утилиты для слайсов (Sort, Contains, Index, ...)
- `maps` — утилиты для map (Keys, Values, Clone, ...)
- `cmp` — Ordered constraint, Compare

### sync.OnceFunc / OnceValue / OnceValues

```go
init := sync.OnceFunc(func() { setup() })
getDB := sync.OnceValue(func() *sql.DB { return connect() })
getDB2 := sync.OnceValues(func() (*sql.DB, error) { return sql.Open(...) })
```

### PGO (GA)

Profile-Guided Optimization теперь стабильна. Автоматически используется `default.pgo`.

### context.WithoutCancel

```go
bgCtx := context.WithoutCancel(ctx) // наследует values, не наследует отмену
```

### context.AfterFunc

```go
stop := context.AfterFunc(ctx, func() { cleanup() })
defer stop()
```

### Другое

- WASI support
- `panic(nil)` теперь оборачивается в `*runtime.PanicNilError`
