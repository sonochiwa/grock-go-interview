# Go 1.22 (Февраль 2024)

**Одно из самых обсуждаемых изменений** — фикс переменной цикла.

## Ключевые изменения

### Фикс переменной цикла for/range

```go
// До Go 1.22: одна переменная v переиспользуется
for _, v := range items {
    go func() {
        fmt.Println(v) // все горутины видят последнее значение!
    }()
}

// С Go 1.22: КАЖДАЯ итерация создаёт новую v
for _, v := range items {
    go func() {
        fmt.Println(v) // каждая горутина видит своё значение ✓
    }()
}
```

Это breaking change, но на практике исправляет баг, а не ломает код.

### Range over integers

```go
for i := range 10 {
    fmt.Println(i) // 0, 1, 2, ..., 9
}
```

### Enhanced routing in net/http

```go
mux := http.NewServeMux()
mux.HandleFunc("GET /users/{id}", handleGetUser)
mux.HandleFunc("POST /users", handleCreateUser)
// Паттерны с методами и path parameters!
```

### Другое

- math/rand/v2 — новый API для генерации случайных чисел
- `slices.Concat`
- Улучшения PGO (devirtualization)
