# Go 1.23 (Август 2024)

## Ключевые изменения

### Range-over-func (итераторы)

```go
// Теперь for-range работает с функциями!
// iter.Seq[V] = func(yield func(V) bool)

func Backward[T any](s []T) iter.Seq[T] {
    return func(yield func(T) bool) {
        for i := len(s) - 1; i >= 0; i-- {
            if !yield(s[i]) {
                return
            }
        }
    }
}

for v := range Backward([]int{1, 2, 3}) {
    fmt.Println(v) // 3, 2, 1
}
```
