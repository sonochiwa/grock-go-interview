# Go 1.24 (Феврал�� 2025)

## Ключевые изменения

### Swiss Table Maps (по умолчанию)

Новая реализация map на основе Swiss Table:
- Open addressing с groups по 16 слотов
- SIMD для параллельного поиска
- 20-50% быстрее для lookup
- API не изменился — прозрачно для кода

### Generic Type Aliases

```go
type MySlice[T any] = []T
type Result[T any] = pkg.Result[T]
// Раньше type aliases не поддерживали type parameters
```

### Weak Pointers

```go
import "weak"
wp := weak.Make(&myObject)
// wp не предотвращает GC
if obj := wp.Value(); obj != nil {
    // объект ещё жив
}
```

Для кешей, дедупликации, canonicalization.

### Finalizers (новый API)

```go
import "runtime"
runtime.AddCleanup(obj, func(ptr *T) {
    // cleanup когда obj собран GC
}, cleanupArg)
// Заменяет runtime.SetFinalizer — проще и безопаснее
```

### Другое

- `os.Root` — ограничение файловых операций директорией (security)
- Улучшения crypto (FIPS 140-3)
- `testing.B.Loop` — упрощённый benchmark loop
- Omitzero для JSON
