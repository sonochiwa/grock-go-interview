# Go 1.25 (Август 2025)

## Ключевые изменения

### Удаление core types из спецификации

Упрощение спецификации языка. Core types были промежуточным концептом для дженериков — теперь правила описаны напрямую.

### testing/synctest (GA)

```go
import "testing/synctest"

// Виртуальное время для тестирования конкурентного кода
func TestTimeout(t *testing.T) {
    synctest.Run(func() {
        ctx, cancel := context.WithTimeout(context.Background(), 5*time.Second)
        defer cancel()
        // synctest управляет виртуальным временем
        // тест завершается мгновенно, не ждёт 5 с��кунд
    })
}
```
