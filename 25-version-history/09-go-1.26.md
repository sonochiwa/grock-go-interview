# Go 1.26 (Февраль 2026)

## Ключевые изменения

### Green Tea GC (по умолчанию)

Generational garbage collector:
- Два поколения: young (nursery) + old
- Minor GC: быстрая сборка молодых объектов
- Major GC: полная сборка (как раньше)
- Меньше CPU на GC для типичных нагрузок
- Полностью прозрачно — код менять не надо

Подробнее: [10-internals/09-gc-green-tea.md](../10-internals/09-gc-green-tea.md)

### Self-referential Generics

```go
// Теперь можно:
type Node[T any] struct {
    Value T
    Next  *Node[T]
}
// Раньше были ограничения на рекурсивные generic типы
```
