# 14. История версий Go

Ключевые изменения в каждой версии. Знание "что когда появилось" помогает на собесах и при работе с legacy кодом.

## Краткая сводка

| Версия | Дата | Главное |
|--------|------|---------|
| 1.18 | Mar 2022 | Дженерики, fuzzing |
| 1.19 | Aug 2022 | Ревизия memory model, atomic types |
| 1.20 | Feb 2023 | PGO, comparable expansion |
| 1.21 | Aug 2023 | min/max/clear, slog, OnceFunc |
| 1.22 | Feb 2024 | Фикс переменной цикла, range over int |
| 1.23 | Aug 2024 | Range-over-func, iter пакет |
| 1.24 | Feb 2025 | Swiss Table maps, generic type aliases |
| 1.25 | Aug 2025 | Удаление core types, synctest GA |
| 1.26 | Feb 2026 | Green Tea GC |
