# Reverse Slice

Напиши функцию `ReverseSlice[T any](s []T) []T`, которая возвращает новый слайс с элементами в обратном порядке. Исходный слайс не должен изменяться.

## Требования

- Функция должна быть generic
- Исходный слайс не мутируется
- Пустой слайс возвращает пустой слайс
- `nil` слайс возвращает `nil`

## Пример

```go
ReverseSlice([]int{1, 2, 3, 4, 5}) // []int{5, 4, 3, 2, 1}
ReverseSlice([]string{"a", "b", "c"}) // []string{"c", "b", "a"}
```
