# Table-Driven Tests

Функция `FizzBuzz(n int) string` уже реализована. Напиши исчерпывающие table-driven тесты:

- Обычные числа → число как строка
- Кратные 3 → "Fizz"
- Кратные 5 → "Buzz"
- Кратные 15 → "FizzBuzz"
- Граничные случаи: 0, отрицательные, 1

В тесте уже есть 1 case — добавь минимум 8 ещё.
